## Sistem Login & Autentikasi

### 1. Login Warga
- **Kredensial**
  - Nomor Anggota (unique identifier)
  - Password
- **Keamanan**
  - Password encryption
  - Session management
  - Rate limiting untuk mencegah brute force
  - OAuth 2.0 support

### 2. Login Majelis
- **Kredensial**
  - Email Majelis
  - Password
  - Bidang Pelayanan
- **Keamanan**
  - Role-based access control
  - Session timeout
  - Login history
  - Multi-factor authentication

### 3. Login Admin
- **Kredensial**
  - Username Admin
  - Password
  - Token Autentikasi (2FA)
- **Keamanan**
  - Two-factor authentication
  - IP whitelist
  - Audit log
  - Comprehensive access monitoring
